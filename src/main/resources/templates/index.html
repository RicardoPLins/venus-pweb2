Vênus

Este é o Vênus, o seu app de colegiado muito mais fácil e rápido.

Como administrador, você poderá:
  • Realizar CRUDs para colegiadosColegiados
  • Realizar CRUDs para alunos Alunos
  • Realizar CRUDs para professores Professores
  • Realizar CRUDs para coordenadores Coordenadores
  • Realizar CRUDs para assuntos de processos Processos
Como aluno, você poderá realizar operações no colegiado:
  • Cadastrar um novo processo
  • Consultar o estado dos processos
Como professor, você poderá realizar operações no colegiado:
  • Consultar os processos
Como coordenador, você poderá realizar operações no colegiado:
  • Consultar o estado dos processos
